Evaluation Method: WebAVALIA

WebAVALIA is a framework that allows teachers to evaluate their students' performance in a classroom. It has been designed for self and peer assessment between students in a group. The assessment can have multiple milestones and students assess their peers and themselves (if self assessment is enabled) for each milestone. Each milestone's self and peer assessment scores are weighted agains their pre determined weightage and put into a formula to calculate the final scores for each student.

Evaluation Method: Quasi-Arithmetic Scoring System

QASS is a math model that allows teachers to evaluate their students' performance in a classroom. It is useful for peer assessment. This model can only have one milestone. It calculates scores based on ratings and contributions
